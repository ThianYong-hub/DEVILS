# Devils Addon

[Meteor Client](https://github.com/MeteorDevelopment/meteor-client) کے لیے ایڈآن: PvP، یوٹیلیٹی اور Nether highway automation۔

## ڈاؤن لوڈ

- موجودہ بلڈ (`v0.0.54`): [jar ڈاؤن لوڈ کریں](https://github.com/ThianYong-hub/DEVILS/releases/download/v0.0.55/devils-addon-0.0.55.jar)
- تازہ ترین ریلیز: [کھولیں](https://github.com/ThianYong-hub/DEVILS/releases/latest)
- تمام ریلیزز: [کھولیں](https://github.com/ThianYong-hub/DEVILS/releases)

## ضروریات

- Minecraft `1.21.11`
- Fabric Loader `0.16.14+`
- Java `21`
- `1.21.11` کے لیے Meteor Client

## انسٹال

1. Meteor Client انسٹال کریں۔
2. اوپر دیے گئے لنک سے jar ڈاؤن لوڈ کریں۔
3. فائل کو `.minecraft/mods` میں رکھیں۔
4. گیم چلائیں اور Meteor میں `Devils` کیٹیگری کھولیں۔

## اہم ماڈیولز

- Combat: `auto-cev`, `auto-pearl`, `tnt-bomber`, `lava-bucket`, `mace-spoof`
- Movement: `auto-wasp`, `anti-wasp`, `h-clip`, `v-clip`
- World: `highway-builder-plus`
- Utility/Player: `auto-anvil-rename`, `tracker-player`, `discord-rpc`

## کمانڈز

- `.autoraname setname <text>`
- `.autoraname clearitems`
- `.example`

## ڈویلپرز کے لیے

```bash
git clone https://github.com/ThianYong-hub/DEVILS.git
cd DEVILS
./gradlew build
```

Windows:

```bat
gradlew.bat build
```

ٹیسٹس:

```bash
./gradlew test
```
